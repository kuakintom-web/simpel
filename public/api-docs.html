SIMPEL-Alkhairaat API Documentation

Dokumentasi lengkap untuk API REST

🔐 Authentication

Semua endpoint memerlukan user session. Login terlebih dahulu sebelum menggunakan API.

📚 School Endpoints

GET
/api/schools
Dapatkan daftar semua sekolah

Response:

{ "status": 200, "message": "Schools retrieved successfully", "data": [...] }
GET
/api/schools/:id
Dapatkan detail sekolah berdasarkan ID

Response:

{ "status": 200, "message": "School retrieved successfully", "data": {...} }
POST
/api/schools
Buat sekolah baru

Request Body:

{ "name": "SMA Alkhairaat", "npsn": "20301234", "district_id": 1, "address": "Jl. Pendidikan No. 123", "phone": "0812345678", "email": "info@alkhairaat.sch.id", "principal_name": "Drs. Ahmad", "total_students": 500, "total_teachers": 50, "founded_year": 2010 }

💰 Financial Endpoints

GET
/api/financial/schools/:id
Dapatkan daftar laporan keuangan sekolah
GET
/api/financial/:id
Dapatkan detail laporan keuangan lengkap dengan transaksi
POST
/api/financial
Buat laporan keuangan baru

Request Body:

{ "school_id": 1, "report_date": "2024-01-31", "opening_balance": 5000000, "closing_balance": 7500000 }

📊 Error Responses

400 Bad Request - Request tidak valid atau parameter kurang
{ "status": 400, "message": "Missing required fields", "data": [] }
401 Unauthorized - User belum login atau session expired
{ "status": 401, "message": "Unauthorized", "data": [] }
404 Not Found - Resource tidak ditemukan
{ "status": 404, "message": "School not found", "data": [] }
500 Internal Server Error - Error pada server
{ "status": 500, "message": "Internal server error", "data": [] }

📝 Usage Examples

JavaScript Fetch

// Get schools fetch('/api/schools') .then(res => res.json()) .then(data => console.log(data)); // Create school fetch('/api/schools', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: 'New School', npsn: '123456', district_id: 1 }) }) .then(res => res.json()) .then(data => console.log(data));
